# 🌞🪐 Moving Solar System

> An interactive and animated Solar System simulation built with HTML, CSS, and JavaScript.

Experience the beauty of space through a visually engaging Solar System model where planets orbit around the Sun with smooth animations and realistic motion effects. This project combines creativity and frontend development to provide an educational and immersive experience.
